patchthis.app:$|

Patchthis.app provides a curated list of CVEs that organizations should prioritize for remediation from the following high-quailty open-source sources:

  • CISA Known Exploited Vulnerabilities Catalog
  • Metasploit Modules
  • EPSS CVEs Above .90 (on a 0.0 to 1.0 scale)

The list is updated hourly and contains over 3400 CVEs. The output is avavilble in this CSV file. The projects code is avavilble in this GitHub Repo.

about:$|

Patchthis.app is an early-stage project by Jerry Gamblin to help collect, combine, and distribute high-quality open-source vulnerability intelligence to help organizations decide how to prioritize their limited patching cycles.
