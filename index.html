PatchThis.app

Patchthis.app provides a combined list of CVEs that organizations should prioritize for remediation from the following open source sources:

  • CISA Known Exploited Vulnerabilities Catalog
  • Metasploit Modules
  • EPSS Vulnerabilities Scoring Above .90 (on a 0.0 to 1.0 scale)

The list are curated now is updated hourly and contains over 3400 CVEs. The output is saved in this CSV file and the code is avavilble in this GitHub Repo.
